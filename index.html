Projects

EEG based study of meditation brain signals

The project involved working on a completely new application domain of machine learning and data analysis. Working with time series data is a challenging task in itself, but in this project the mode of input was electrical signals. Different from other time-series data, electrical signals require very domain-specific processing. Further, EEG signals are very low voltage signals which are prone to noise from multiple sources. Therefore, the processing phase required rigourous research of the methods used for filtering, and cleaning of EEG signals. Finally a pipeline to process signals was built using EEGLAB over MATLAB. Further, temporal and spectral features were extracted from the filtered signals. The extracted signals were tested for inter-state differences using statistical tests. Further, machine learning was used to automatically classify the brain states using the features extracted from the brain signals.

Question generation using T5

This project found a novel way to use the Hugging face's Text-to-text-transfer transformer (T5), which was one of the first multi-task transformer by Hugging face, way before GPT was capable enough to become 'the ChatGPT'. The project involved working with multiple datasets in order to create different types of questions, ranging from Short-answer type questions (using Stanford's SQuAD dataset, which we inverted for our application), long-asnwer type questions (), multiple-choice type questions, fill-in-the-blanks type questions, and true-or-false type questions (using BoolQ). This required training multiple models.
